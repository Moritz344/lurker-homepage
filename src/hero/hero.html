Lurker

opensource chat client for twitch.tv

Download for Linux

Features

Chat

View chat messages. View badges and emotes. See user colors for chatters. Click on any username to open their user profile card

Stream Information

View stream thumbnail. See current viewers. See stream title.

Other Features

Switch Channels. See Channels you follow. Open Mod View. Open Stream in Browser. Clear Chat. Disconnect from Stream.
